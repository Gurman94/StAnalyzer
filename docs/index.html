Key Features

Dual solver engines

Truss solver for determinate pin-jointed systems. Beam or frame solver for continuous beams and rigid frames.

Real time results

Support reactions, internal forces, shear and moment diagrams. Deflection visualization and moment tracing.

Shape library

0 plus AISC steel shapes - W, HSS, C, L and more. Custom sections planned.

Truss or frame checks

Built in determinacy check using m + r versus 2j. Classifies determinate, indeterminate or unstable.

Stress module

Axial, shear, bending stresses planned for the next release after cost optimization.
